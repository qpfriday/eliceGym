# 주요 사항
## 1. 프로젝트 개요
- frontend: vue.js
- backend: Spring Boot
- database: mariaDB

## 2. 현제 구현 상태
1. 메인 페이지에 card 로 상품 진열
2. Header, Footer 구현 (로그인 상태에 따라 구성 요소 바뀜)
3. jwt 를 이용한 로그인, 로그아웃 구현
4. 장바구니에 담긴 상품 리스트 구현
5. 구매하기 누르면 장바구니안에 담긴 상품들을 구매
6. 주문 조회를 누르면 주문한 상품 볼 수 있음

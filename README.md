# eliceGym
> 헬스 케어와 관련된 스포츠 의류, 스포츠 용품, 보조제 등
여러가지 스포츠 용품을 사고 팔 수 있는 쇼핑몰 개발<br>
> 개발기간: 24.03.25~24.04.20(4주)

# 배포 주소
[http://34.64.52.132](http://34.64.138.49:3001/)

# 프로젝트 소개
- 헬스 관련 용품을 사고 팔 수 있는 쇼핑몰 사이트입니다
- 판매자는 이미지를 포함해 상품 정보를 입력해서 등록할 수 있습니다
- 구매자는 상품들을 장바구니에 저장해서 한꺼번에 주문할 수 있습니다

# Team4-Back
> 스팀 게임 아카이빙 및 커뮤니티 사이트<br>
> 개발기간: 24.05.20~24.06.14(4주)

# 배포 주소
http://34.64.52.132

# 프로젝트 소개
1. 스팀 에서 제공하는 API 활용 (https://steamapi.xpaw.me/)
2. 스팀 로그인을 포함한 소셜 로그인 기능 구현
3. 커뮤니티 게시판 구현 (자유, 리뷰, 공략, 토론 등)
4. 스팀 상점에 있는 상품 불러오기
5. 내 스팀 게임 목록 및 업적 달성률 조회
